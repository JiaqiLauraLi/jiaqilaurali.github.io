Clockify: Data Warehousing Project

In today’s increasingly complex and hybrid business environment, companies are seeking new ways of tracking employees’ work time and efficiency through data. With this goal in mind, we helped Data Sleek make the best use of Clockify’s data to diagnose and understand employee performance, and to empower more informed business decisions.

In this project, we have:

  • Queried data from Data Sleek’s cloud data warehouse using Snowflake;
  • Analyzed, defined, and developed 8 KPIs covering 4 business aspects: time-tracking, financials, employee performance, and time management;
  • Put together an interactive, customizable dashboard displaying real-time information in Tableau.

    Image Classification Project

    If given camera permission from users, what can app developers do to improve their products? Will facial recognition and emotion detection help product owners better understand user preferences as they navigate through different parts of the app? Our project mainly focuses on exploring the initial stage of this idea: facial recognition and emotion-based recommendations.

    In this project, we have:

  • Developed an image classification model using Python TensorFlow and Scikit-Learn for facial recognition and emotion detection;
  • Built an interactive web app that recommends media contents based on analyses of user uploaded figures;
  • Discussed restrictions, limits, and potential privacy issues that this technology may cause.

    Network Link Prediction Project

    Link prediction problem has been a heated topic in the field of network analysis. Given timestamp information and a network of interactions between different users, we are interested in learning whether it is possible to predict if a new interaction will exist between two users in the near future.

    In this project, we have:

  • Explored different network-related metrics using Facebook network data with 300K edges and 50K nodes;
  • Examined, fine-tuned, and compared the performance of multiple machine learning models (Logistic Regression, SVM, GBM, etc.) to predict new interactions based upon current network data.
